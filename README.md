# 🚀 AI-Powered Research & Patent Discovery Platform

## 📌 Project Overview

The AI-Powered Research & Patent Discovery Platform is a web-based application designed to help users explore research topics, discover patent opportunities, analyze technology trends, and generate research insights through an interactive dashboard.

This platform provides user authentication, research analytics, patent discovery features, and a modern user interface built using Flask and SQLite.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/pirtikshas-coder/ai-research-patent-platform.git
```

### Navigate to Project

```bash
cd ai-research-patent-platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
